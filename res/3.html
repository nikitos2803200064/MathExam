Число $b$ называется пределом функции $y=f(x)$ при $x\rightarrow+\infty$, если, каково бы ни было положительное число $\varepsilon$, можно найти такое число $N$, что для всех $x$, больших $N$, выполняется неравенство $\vert f(x)-b\vert <\varepsilon$.
Число $b$ называется пределом функции $y=f(x)$ при $x\rightarrow-\infty$, если, каково бы ни было положительное число $\varepsilon$, можно найти такое число $M$, что для всех $x$, меньших $M$, выполняется неравенство $\vert f(x)-b\vert <\varepsilon$.
Число b называется пределом функции $y=f(x)$ при $x\to \infty$, если для любого положительного числа $\varepsilon$ существует положительное число $\delta$, зависящее от $\varepsilon$ ($\delta =\delta \left( \varepsilon \right)$) такое, что для всех значений аргумента из области определения функции, больших по абсолютной величине этого числа $\delta$, значения функции $y=f(x)$ отличаются по величине от указанного числа $b$ меньше, чем на $\varepsilon$: $$\lim_{x\rightarrow\infty}f(x)=a\Leftrightarrow(\forall\varepsilon>0\;\exists\delta=\delta(\varepsilon)>0\;:\;\forall x\in D(y)\;:\;\vert x\vert>\delta\;\vert f(x)-a\vert<\varepsilon)$$
